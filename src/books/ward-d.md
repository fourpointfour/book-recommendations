## Why I recommend this book

First Freida McFadden book I stumbled after reading And Then There Were None by Agatha Christie and I have to say, this is some seriously good storytelling.

More points to recommend this book would be:
- Crazy plot twists.
- Even better character build-ups.
- Very thrilling environment set up as well.
